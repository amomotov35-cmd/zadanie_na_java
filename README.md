# Приложение "Корзина покупок"

Консольное приложение для управления корзиной покупок с системой логирования операций.

## Функциональность

- Добавление товаров в корзину
- Удаление товаров из корзины
- Просмотр содержимого корзины
- Расчет общей суммы покупок
- Автоматическое логирование всех операций

## Структура проекта

- `Product` - класс товара (название, цена)
- `Cart` - класс корзины (хранение товаров, операции)
- `LogEntry` - класс для записи логов операций
- `Main` - основной класс с примером использования

## Требования

- Java 17 или выше
- Компилятор Java (javac)
- Терминал/командная строка

## Инструкция по запуску

### Способ 1: Через командную строку

1. Сохраните код в файл `Main.java`
2. Откройте терминал в папке с файлом
3. Скомпилируйте программу:
   ```bash
   javac Main.java
